# Lezione7

## Ambito
- Mini progetto Todo List / ledger con Bootstrap installato da npm.
- È una pagina statica servita localmente, non un'app con backend.

## Regole operative
- Mantieni coerenti `todolist.html`, CSS e JS della lezione.
- Non rompere gli id e le classi usati dallo script per il rendering e gli eventi.
- Conserva il salvataggio in `localStorage` e il comportamento di filtro/stato attività.

## Comandi utili
```bash
npm install
npm run start
```

## Verifica
- Apri `http://localhost:5500/todolist.html` dopo l'avvio del server.
- Controlla che Bootstrap e gli asset locali vengano caricati correttamente.
