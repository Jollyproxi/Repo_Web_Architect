# Todo List Ledger (Bootstrap via npm)

Mini progetto Todo List con interfaccia ledger usando Bootstrap installato da npm.

## Requisiti

- Node.js 18+

## Avvio rapido

1. Installa le dipendenze:

```bash
npm install
```

2. Avvia un server locale nella cartella `HTML/Lezione7`:

```bash
npm run start
```

3. Apri nel browser:

- `http://localhost:5500/todolist.html`

## Funzionalita

- Aggiunta attivita
- Cambio stato (aperta/completata)
- Eliminazione attivita
- Filtri: tutte, da fare, completate
- Salvataggio automatico su `localStorage`
