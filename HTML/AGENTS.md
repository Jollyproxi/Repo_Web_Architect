# HTML

## Ambito
- Lezioni e demo statiche di HTML/CSS/JS.
- Ogni cartella `Lezione*` è pensata per essere aperta direttamente nel browser.

## Regole operative
- Non introdurre backend, build system o dipendenze non necessarie.
- Mantieni esempi semplici e didattici, senza refactor ampi fuori dalla lezione interessata.
- Preserva i riferimenti relativi ad asset, immagini e CSS già usati nelle pagine.

## Caso speciale
- `Lezione7/` usa Bootstrap da npm e richiede avvio locale con server statico.
- Per quella cartella segui anche `HTML/Lezione7/AGENTS.md`.

## Verifica tipica
- Apri l'`index.html` della lezione nel browser o usa Live Server.
- Se una lezione ha un comando dedicato, documentalo nel suo `AGENTS.md` locale.
