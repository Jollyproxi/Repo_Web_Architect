# esempio1

## Ambito
- Esempio base di SCSS con struttura `scss/` e `css/` e una pagina `index.html`.

## Regole operative
- Non introdurre build system obbligatori: mantieni la possibilità di aprire `index.html` direttamente.
- Se modifichi gli scss, aggiorna anche `css/main.css` (se presente) o documenta il comando di compilazione.

## Comandi utili (opzionali)
```bash
sass scss/main.scss css/main.css
```

## Verifica tipica
- Apri `index.html` nel browser e verifica che gli stili siano applicati.
