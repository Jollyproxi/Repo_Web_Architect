# HTML/Lezione8_SCSS/esempio3_davide/scss

## Ambito
Sorgenti SCSS per l'esempio `esempio3_davide` della Lezione 8.

## Regole operative
- Modificare gli SCSS e rigenerare il CSS invece di editare i file CSS compilati.
- Mantenere variabili e mixin condivisi in `_variables.scss` e `_mixins.scss`.

## Comandi utili
- Se è presente un `package.json` con script di build, eseguirlo per compilare gli SCSS (es. `npm run build` o `npm run scss`).
- In mancanza di script: usare un compilatore SCSS locale (es. `sass input.scss output.css`).

## Verifica tipica
- Aprire `HTML/Lezione8_SCSS/esempio3_davide/index.html` dopo la compilazione per verificare il risultato.
