# Lezione8_SCSS

## Ambito
- Raccolta di esempi per l'uso di SCSS (esempio1, esempio2, esempio3, esempio3_davide).

## Regole operative
- Gli esempi sono pensati per essere aperti direttamente nel browser; non è obbligatorio un tool di build.
- Se aggiungi partial SCSS condivisi, documentali nel `AGENTS.md` locale della cartella che li contiene.
- Mantieni i CSS compilati compatibili con il `index.html` presente in ciascun esempio.

## Comandi utili (opzionali)
- Se usi Dart Sass localmente, un comando tipico per compilare uno degli esempi è:

```bash
sass scss/main.scss css/main.css
```

## Verifica tipica
- Apri `esempio*/index.html` nel browser.
