Prova fetch

Testare tutti i verbi dello HTTP con la fetch API, chiamando l'API di json-server.

  • GET -> ricevi tutti gli oggetti o le collection
  • GET -> ricevi oggetti filtrati
  • POST -> crea oggetti all'interno di un collection
  • PUT -> modifica integralmente un oggetto di una collection
  • PATCH -> modifica parzialmente un oggetto di una collection
  • DELETE -> elimina un oggetto

    • Ogni singolo metodo deve avere un pulsante al quale è collegato. Nel caso della POST utilizza dei campi input per inserire le informazioni necessarie alla creazione dell'oggetto.

    BONUS: inserisci uno spinner (fai riferimento agli spinner di bootstrap) quando carichi un nuovo oggetto attraverso la POST
