# projectBeatles

Landing page e pagine dedicate per la band The Beatles.

Setup rapido:

1. Installare dipendenze per SCSS (se non presenti):

```powershell
npm install
```

2. Compilare SCSS in CSS:

```powershell
npm run build:scss
# o in sviluppo
npm run watch:scss
```

3. Aprire `index.html` nel browser.

Note:
- Il progetto usa la MusicBrainz API per ottenere la discografia. Se la API non è raggiungibile, i dati locali verranno usati come fallback.
- Non è richiesto alcun backend.
