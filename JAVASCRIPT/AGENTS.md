# JAVASCRIPT

## Ambito
- Lezioni, demo ed esercitazioni JavaScript, DOM, funzioni e OOP.
- La maggior parte dei materiali è statica e si apre direttamente nel browser.

## Regole operative
- Mantieni gli esempi semplici, didattici e facilmente leggibili.
- Evita refactor trasversali tra lezioni diverse: lavora nella cartella che contiene il materiale richiesto.
- Non introdurre backend o dipendenze extra se il compito non lo richiede.
- Se tocchi esercizi come `projectRubrica/` o `projectTodoList/`, resta coerente con la loro struttura locale.

## Verifica tipica
- Apri l'HTML interessato nel browser o con Live Server.
- Se la modifica riguarda solo JS didattico, verifica che l'esempio resti eseguibile senza build.
