# projectTodoList

## Ambito
- Todo list didattica con HTML, CSS e JavaScript puro.
- Nessun backend: tutto è gestito nel browser.

## Regole operative
- Mantieni semplice la UI con input, bottone e lista attività.
- Conserva il collegamento tra `index.html`, `css/` e `js/`.
- Evita dipendenze o architetture aggiuntive se non richieste.
- Se cambi il comportamento delle attività, verifica che il DOM resti coerente con gli id già presenti.

## Verifica tipica
- Apri `index.html` nel browser.
- Controlla aggiunta, visualizzazione e interazione con gli elementi della lista.
