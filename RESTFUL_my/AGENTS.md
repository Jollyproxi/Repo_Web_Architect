**⚠️ REGOLA FISSA:** Prima di procedere, leggi **SEMPRE** il `../AGENTS.md` di root e tutti gli AGENTS.md lungo il percorso fino a questa cartella.

## Ambito
- Variante personale dei materiali RESTful con dati di esempio nella cartella `dbjson/`.
- Include anche progetti backend completi (API Express) per esercitazioni.

## Regole operative
- Mantieni i JSON validi e la struttura coerente con gli esempi locali.
- I progetti backend (`taskflow-api/`) hanno la propria `AGENTS.md` con regole specifiche: segui quella prima di questa.
- Per i dati statici (`dbjson/`), mantieni la natura statica del materiale.
- Se aggiungi file, resta allineato al formato e allo stile già usati nella cartella.

## Verifica tipica
- Valida sintassi e struttura dei dati JSON.
- Per i progetti backend, verifica che `npm run dev` avvii il server senza errori.

## Indice AGENTS.md
- `taskflow-api/AGENTS.md` — backend Express per TaskFlow (CRUD, JWT, validazione)
- `taskflow-frontend/AGENTS.md` — frontend vanilla JS che consuma l'API TaskFlow
